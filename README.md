# Smart Classroom Monitoring System

An IoT-based classroom monitoring system using ESP32 that automates attendance, monitors environmental conditions, and enhances safety.

## Features
- Occupancy detection using IR sensors
- Temperature monitoring using DHT11
- Gas detection using MQ2 sensor
- Automatic light and fan control
- LCD display with real-time alerts
- Buzzer alert system

## Components Used
- ESP32
- DHT11 Sensor
- MQ2 Gas Sensor
- IR Sensors
- LCD Display
- Buzzer and LEDs

## Working
The system counts the number of people entering and exiting using IR sensors. Based on occupancy and temperature, lights and fans are automatically controlled. Gas detection or high temperature triggers alerts using buzzer and LED. All data is displayed on an LCD.
